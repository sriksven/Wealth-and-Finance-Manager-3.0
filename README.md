# Wealth and Finance Manager 3.0

A comprehensive tool for managing personal finances, tracking expenses, and visualizing wealth data. This project leverages **React (Vite)** for a responsive frontend and integrates with **Firebase** for real-time data and authentication.

## Features

-   **Dashboard Overview**: Visualize income, expenses, and net worth.
-   **Transaction Tracking**: Log and categorize your financial transactions.
-   **Recurring Items**: Manage subscriptions and recurring bills.
-   **Data Visualization**: Interactive charts using Chart.js.
-   **Secure Authentication**: Powered by Firebase Authentication.
